## Using venv

Using a virtual environment (venv) helps isolate projects and their dependencies. Without virtual environments, there could be many conflicts between library versions. For example, if Project A requires one version of a Python library and we create Project B that requires a different version of the same library, it could cause problems for Project A. By using virtual environments, each project lives in its own isolated environment with its own dependencies.



## Face detection vs face recognition

Face detection models are trained to detect faces in an image, but they are not designed to recognize whose face it is. Face recognition models, on the other hand, are trained to identify or verify the identity of a specific person.



We used OpenCV's pre-trained models for face detection.



## Embedding

A computer cannot understand a human face in the way humans do, it requires a numerical representation of the image. Neural networks are trained on millions of images, and they process them as vectors. Similar images correspond to similar vectors.
